# site

Next.js 14 App Router scaffold for the public LF documentation site.

v0.1 scope: landing page, three placeholder routes
(`/concepts`, `/tropes`, `/cli`). Auto-rendering from `src/concepts/`
and `src/tropes/` MDX sources is follow-up work.

## Scripts

- `bun run dev` — dev server on port 3000.
- `bun run build` — production build.
- `bun run start` — run the production build.
- `bun run typecheck` — `tsc --noEmit`.
