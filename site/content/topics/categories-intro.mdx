A *category* is a closed vocabulary: an explicit list of named
members plus optional morphisms (relations or transitions). Before
a value can be used in prose or code, the value must be a declared
member of its category.

This page assembles the `category` Concept, the `category` Trope
(LF's canonical realisation), and the four base members LF ships:
`tags`, `adr-status`, `goal-status`, and `goal-category`. Each
member exports both an Effect Schema (for type-checked use in TS
code) and an MDX prose body for the docs and the vendored
`.literate/`.
