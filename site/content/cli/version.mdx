# `literate version`

Print the LF CLI version and the bundled Trope catalog (id, version,
realising Concept).

## Synopsis

```
literate version
```

## Sample output

```
literate v0.1.0

Bundled Tropes:
  corpus@0.1.0          realises corpus
  session@0.1.0         realises session
  session-start@0.1.0   realises trope
  session-end@0.1.0     realises trope
  decisions@0.1.0       realises decisions
  category@0.1.0        realises category
  term@0.1.0            realises term
  chapters@0.1.0        realises chapters
  adr-flow@0.1.0        realises trope
  goal-flow@0.1.0       realises trope
```
