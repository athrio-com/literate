# `dispositional-domain` — Concept seed

A meaningful application slice scoped within a Layer — the unit
of granularity at which LFMs are authored. One LFM per
Layer-Domain pair. Names are freeform, lowercase, dash-separated;
the same name (e.g. `ux`) can recur in multiple Layers because
the Layer scopes the namespace. A good Domain holds one
coherent slice of current-state and would change for one
coherent reason.
