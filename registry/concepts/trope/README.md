# `@literate/concept-trope`

The meta-Concept seed for Tropes. Declares what a Trope *is* in
LF's substrate; required for `literate learn trope` to resolve.

- `index.ts` — the typed seed (`ConceptOfTrope`,
  `TropeInstanceSchema`).
- `concept.mdx` — the prose body the agent reads.
- `SEED.md` — authoring metadata.
