# `implication` — Concept seed

A typed *soft Goal*: something a session surfaced that *might*
become a Goal — here, in a future session, or not at all.
Closed four-status vocabulary
(`Surfaced | Promoted | Filed | Dismissed`); rationale is
Schema-required when status is `Dismissed`. The `session-end`
Trope refuses to close on a non-terminal Implication.
