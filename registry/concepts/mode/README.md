# `mode` — Concept seed

The operational-stance Concept. Closed three-value vocabulary
(`Exploring | Weaving | Tangling`) naming *how* the work is
being done. Bound to **IMP-N** (the Mandatory Agent
Instruction that overrides agentic-IDE training defaults so
an agent in `Exploring` Mode does not collapse exploration
into premature implementation). Orthogonal to Disposition.
