# `@literate/concept-concept`

The meta-Concept seed. Declares what a Concept *is* in LF's
substrate; required for `literate learn concept` to resolve.

- `index.ts` — the typed seed (`ConceptOfConcept`,
  `ConceptInstanceSchema`).
- `concept.mdx` — the prose body the agent reads.
- `SEED.md` — authoring metadata.
