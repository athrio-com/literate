# `index` — Trope seed

Produce a permanent navigation index at
`corpus/manifests/index.md` summarising every LFM in the corpus
with soft-link references. The index is pure navigation —
carries no decisive content of its own — and is regenerated on
every invocation. Optional but default-shipped: a consumer with
no need for a single navigation file can omit it from their
template, but the `minimal` template seed list includes it so
most consumers discover it without opting in.
