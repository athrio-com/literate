A *corpus* is the prose half of every LF-using repo. It contains
the authored prose that defines what the product is and becomes:
decisions, specs, chapters, sessions, categories, concepts, memos.
Code is derived from this prose; the corpus is the source.

Below: the `corpus` Concept declares the directory shape and
invariants; the `corpus` Trope realises it in a specific layout
that LF ships, with conventions for indexes and mutability.
