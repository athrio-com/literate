# `literate init [template]`

Scaffold a new consumer repo. Copies a template package's tree into
the target directory and runs `compile` to materialise
`.literate/`.

## Synopsis

```
literate init [template] [--target <dir>]
```

| Argument | Default | Description |
|---|---|---|
| `template` | `minimal` | Template id. v0.1 ships `minimal`. |
| `--target`, `-t` | `.` | Target directory (default: cwd). |
