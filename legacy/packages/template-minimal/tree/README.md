# My Literate Project

Scaffolded from `literate init minimal`.

## Layout

- `corpus/` — your authored prose (decisions, categories, sessions,
  and additional folders as Tropes are added).
- `src/` — your derived code and content.
- `.literate/` — vendored LF snapshot. Managed by the `literate`
  CLI. Do not hand-edit.
