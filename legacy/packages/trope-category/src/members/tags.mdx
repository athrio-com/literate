# Tags (base vocabulary)

Initial tag set for decisions. Consumers receive this as a starting
vocabulary when they install the `category` Trope; extending it is a
gated category revision in the consumer's own
`corpus/categories/tags.md`.

## Members

- `#process` — authoring workflow (sessions, gating, review).
- `#protocol` — the LITERATE.md product surface.
- `#corpus` — conventions for `corpus/`.
- `#tooling` — CLI, core library, build.
- `#algebra` — the Concept / Trope / Authored-instance model.
- `#licensing` — licensing decisions.
- `#release` — versioning, publishing.

These seven are deliberately minimal. Consumers grow the tag set
as their domain demands.
