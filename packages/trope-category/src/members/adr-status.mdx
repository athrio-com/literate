# ADR Status (base vocabulary)

Closed set of `Status:` values an ADR can carry.

## Members

- `Open` — drafted, presented for review, not yet decided.
- `Accepted` — in force.
- `Deferred` — deliberately postponed; context preserved.
- `Superseded by ADR-NNN` — replaced by a later ADR.

## Transitions

```
Open → Accepted
Open → Deferred
Open → (deleted on Reject)
Accepted → Superseded by ADR-NNN
Deferred → Accepted (rare)
```
