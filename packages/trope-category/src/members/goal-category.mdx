# Goal Category (base vocabulary)

Closed set of `Category:` values a session-Goal can carry.

## Members

- `exploration` — open-ended investigation; lighter gating.
- `feature` — new user-visible behaviour; requires upstream spec.
- `bugfix` — spec drift correction; spec updated as part of fix.
- `refactor` — internal restructuring with no observable change.
- `prose` — authored work with no code.
- `process` — Literate-workflow change; Protocol or vocabulary edit.
- `migration` — vendor/implementation change; tight scope.

Categories are disjoint at the Goal level: one Goal, one Category.
